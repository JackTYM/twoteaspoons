Fraunces vs Playfair Display

  • Fraunces: Softer, rounder serifs with a friendlier, more approachable feel. Slightly quirky personality.
  • Playfair: More traditional, elegant serifs. Classic recipe book aesthetic.
  • Both work well - Fraunces feels warmer, Playfair feels more refined.
